## [사업계획서] High-Five Focus: 실행력을 파는 5-슬롯 타이머 서비스

### 1. 서비스 개요

**"전략적인 포기와 압도적 집중"**을 슬로건으로 하는 하이파이브 포커스는 사용자의 하루 할 일을 딱 **5개로 제한**하고, 각 할 일마다 **전용 타이머**를 할당하여 시각적 압박과 성취감을 동시에 제공하는 실행 관리 솔루션(Web/App/Widget)입니다.

### 2. 문제점 (Pain Points)

* **할 일 목록의 과부하:** 너무 많은 To-Do 리스트는 시작도 하기 전에 사용자를 무력하게 만듭니다. (Decision Paralysis)
* **시간 감각의 부재:** 단순히 '무엇을 할지'는 알지만, '언제까지 끝낼지'에 대한 시간 관리(Time Boxing)가 부족해 마감 직전까지 일을 미룹니다.
* **분산된 환경:** PC 작업 중에는 스마트폰이 방해 요소가 되고, 스마트폰을 볼 때는 PC의 작업 흐름을 놓치는 등 환경 간 동기화가 부족합니다.

### 3. 문제 해결 방법 (Solutions)

* **5-Slot 제약 시스템:** 인간의 단기 기억과 집중력을 고려해 하루 딱 5개의 핵심 과제만 등록 가능하게 하여 우선순위 설정을 강제합니다.
* **프로그레스 타이머 위젯:** 모바일 홈 화면과 데스크탑 상단에 실시간으로 줄어드는 시간 바(Bar)를 노출하여 '마감 효과(Deadline Effect)'를 극대화합니다.
* **게임화된 보상/패널티:** * **보상:** 완수 시 캐릭터 성장, '포커스 코인' 지급, 상위 % 집중력 리포트 발행.
* **패널티:** 미완료 시 캐릭터의 외형 변화(노화/우울), 예치금 차감, 혹은 다음 날 할 일 슬롯 1개 잠금(Penalty Box).



### 4. 비즈니스 모델 (Monetization)

| 모델 구분 | 상세 내용 |
| --- | --- |
| **Freemium (구독)** | 기본 기능 무료, 위젯 커스터마이징 및 상세 통계 리포트는 월 구독(Pro) 적용. |

### 5. 약점 (Weaknesses)

* **유연성 부족:** 갑작스러운 업무 추가가 많은 직군(예: CS, 비서)에게는 5개 슬롯 제한이 제약으로 느껴질 수 있습니다.
* **초기 진입 장벽:** 할 일마다 예정 시간을 입력해야 하는 과정이 귀찮음(Friction)으로 작용할 수 있습니다.
* **경쟁 포화:** 이미 시장에 수많은 To-Do 및 타이머 앱이 존재하여 확실한 '브랜드 색깔'이 필요합니다.

# [PRD] High-Five Focus 제품 요구사항 정의서

## 1. 프로젝트 개요

* **프로젝트 명:** High-Five Focus (하이파이브 포커스)
* **목적:** 사용자가 하루 5개의 핵심 과업에만 집중하게 하여, 시간 관리 효율을 극대화하고 실행력을 높이는 타임박싱(Time-Boxing) 도구 구축.
* **핵심 가치:** * **집중(Focus):** 5개 슬롯 제한을 통한 선택과 집중.
* **시각화(Visual):** 타이머와 프로그레스 바를 통한 잔여 시간의 직관적 노출.
* **지속성(Consistency):** 게임화된 보상과 패널티를 통한 습관 형성.


* **타겟 유저:** 갓생을 지향하는 대학생, 재택근무 프리랜서, 마감 기한 준수가 힘든 직장인.

## 2. 유저 플로우 (User Flow)

1. **온보딩:** 앱 실행 → 서비스 컨셉(5개 슬롯) 안내 → 소셜 로그인.
2. **할 일 등록:** 메인 화면의 비어있는 5개 슬롯 클릭 → 할 일 내용 및 예정 소요 시간(예: 60분) 입력.
3. **타이머 시작:** 특정 할 일의 'Start' 버튼 클릭 → 타이머 카운트다운 시작 → 위젯/데스크탑에 실시간 동기화.
4. **진행 중:** 예정 시간의 50%, 80%, 100% 시점에 푸시 알림 수신 (리마인드).
5. **완료 및 보상:** 할 일 완료 버튼 클릭 → 캐릭터 경험치 획득 및 코인 지급 → '하이파이브' 애니메이션 노출.
6. **하루 마무리:** 5개 과업 달성률 리포트 확인 및 SNS 공유.

## 3. 핵심 기능 (Key Features)

### 3.1 5-슬롯 타임박싱 시스템

* **기능:** 하루 최대 5개의 할 일만 등록 가능.
* **상세:** 5개가 꽉 차면 추가 등록 불가 (삭제 후 등록은 가능하나 패널티 부여 검토).

### 3.2 실시간 동기화 타이머 & 위젯

* **기능:** 웹-앱-위젯 간 실시간 타이머 연동.
* **상세:** * **모바일 위젯:** 홈 화면에서 현재 작업의 남은 시간을 프로그레스 바 형태로 표시.
* **웹 대시보드:** 브라우저 탭 아이콘에 남은 시간 표시(Favicon Timer) 및 전체 진행 상황 시각화.



### 3.3 스마트 리마인드 알림

* **기능:** 진행 상황에 따른 동적 메시지 발송.
* **상세:** "예상 시간보다 10분 지났습니다. 일정을 조정할까요?" 등의 대화형 알림.

### 3.4 게이미피케이션 (보상/패널티)

* **보상:** 연속 달성 시 캐릭터 진화, 테마 구매용 코인 지급.
* **패널티:** 시간 초과 시 캐릭터 우울 상태 변화, '집중 예치금' 모드 시 포인트 차감.

## 1. 핵심 UX 전략: "인지 부하의 최소화"

이 서비스의 성공 비결은 사용자가 '무엇을 할지' 고민하는 시간을 줄이고 '당장 실행'하게 만드는 데 있습니다.

* **진행 상태의 컬러 코딩 (Traffic Light UX):**
* **초록색:** 정상 진행 중 (남은 시간 50% 이상)
* **노란색:** 주의 필요 (남은 시간 20% 미만)
* **빨간색:** 시간 초과 (패널티 발생 직전 혹은 발생 중)
* *효과:* 글자를 읽지 않아도 색상만으로 본인의 위급 상황을 즉각 인지합니다.


* **원터치 타임 스위칭 (Seamless Transition):**
* 1번 할 일을 하다가 2번을 클릭하면, 1번 타이머는 자동으로 일시 정지(Pause)되고 2번이 즉시 시작됩니다. 여러 번 클릭할 필요 없는 '싱글 액션' 설계입니다.


* **햅틱 피드백 (Mobile Only):**
* 타이머 종료 1분 전이나 완료 시, 기분 좋은 진동(Haptic)을 통해 촉각적 성취감을 제공합니다.

## 2. 권장 사용자 흐름 (Detailed User Flow)

사용자가 앱을 켜서 하루를 마칠 때까지의 여정입니다.

### ① 단계: 아침의 결단 (Planning)

* **Flow:** 앱 진입 → 비어있는 5개 슬롯 확인 → 할 일 입력 → **예상 소요 시간 설정** (휠 다이얼 방식 추천).
* **UX 포인트:** 5개를 다 채우지 않아도 시작할 수 있지만, 5개가 넘어가면 "오늘의 에너지는 5개면 충분합니다"라는 넛지(Nudge) 메시지 노출.

### ② 단계: 몰입의 시작 (Execution)

* **Flow:** 'Start' 버튼 클릭 → 스마트폰 화면 잠금 → **잠금 화면 위젯(Live Activity)**에서 실시간 타이머 작동.
* **UX 포인트:** 폰을 켜지 않아도 잔여 시간을 볼 수 있게 하여 스마트폰 중독(딴짓)을 방지합니다.

### ③ 단계: 위기와 조정 (Crisis Management)

* **Flow:** 작업이 예상보다 길어짐 → 타이머 빨간색 변화 → **'시간 추가'** 버튼 혹은 **'다음 일정 밀기'** 기능 사용.
* **UX 포인트:** 시간을 추가할 때는 '포커스 코인'이 소모되거나 캐릭터가 살짝 실망하는 표정을 지어 '약한 패널티'를 줍니다.

### ④ 단계: 하이파이브 보상 (Success)

* **Flow:** 작업 완료 버튼 클릭 → 화면 전체에 **'하이파이브' 애니메이션** 노출 → 캐릭터 경험치 상승 → 완료된 슬롯이 황금색으로 변경.
* **UX 포인트:** 단순한 체크 표시보다 화려한 시각 효과를 주어 도파민 분비를 촉진합니다.

### ⑤ 단계: 마감 및 회고 (Reflection)

* **Flow:** 5개 완료 혹은 하루 종료 → **'오늘의 포커스 리포트'** 자동 생성 → SNS 공유 버튼 활성화.
* **UX 포인트:** "당신은 오늘 300분을 집중했습니다. 이는 어제보다 15% 향상된 수치입니다"와 같이 구체적인 데이터 비교를 제공합니다.

* **데스크탑 웹:** 브라우저 상단 탭(Favicon)에 숫자가 줄어드는 애니메이션을 넣어, 다른 페이지를 보고 있어도 압박감을 느끼게 합니다.
* **모바일 위젯:** 위젯 내에서 바로 '완료' 버튼을 누를 수 있는 인터랙티브 위젯 기능을 활용합니다. (앱을 굳이 열지 않아도 됨)
